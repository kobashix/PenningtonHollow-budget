TRY IT OUT
═══════════════════════════════════════════════════════════════════════════════

1. Find any row in the Schedule of Values table
2. Click the "Log" button
3. A modal appears:
   - Amount: Enter cost (e.g., 5000)
   - Date: Defaults to today
   - Type: Choose "Owner's Cash Equity" or "Construction Loan Draw"
   - Note: Optional (e.g., "Invoice #ABC123")
4. Click "Log Payment"
5. Dashboard updates instantly!
   - % Complete changes
   - Cash/Finance remaining updates
   - Charts visualize the new data
   - Variance calculated
